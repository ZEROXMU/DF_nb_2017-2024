# DF_nb_2017-2024
Epidemiological Study Dataset of Dengue Fever in Ningbo, China (2017-2024)

## Dataset Overview
This dataset compiles aggregated and anonymous surveillance data on dengue fever epidemiology, mosquito vector density, transmission capacity indicators, and related visual analysis for Ningbo, Zhejiang Province, from 2017 to 2024. It is intended to support public health research, policy development, and disease prediction efforts.

## Data Content
### 1. Case Surveillance Data
Time Period: January 1, 2017 - December 31, 2024
Coverage Area: All districts of Ningbo Municipality

### 1. Key epidemiological investigation information: 
Intensity of Public Health and Social Measures (PHSMs)
Time Period: January 1, 2017 - December 31, 2024
Coverage Area: Citywide

### 3. Vector Density Surveillance Data
Monitoring Sites: Breteau Index (BI) of all districts of Ningbo Municipality
Time Period: April 1, 2017 - November 31, 2024

### 4. Transmission Capacity Indicators
Basic Reproduction Number (R0)
Time Period: January 1, 2017 - December 31, 2024
Coverage Area: Citywide

## File Structure
1. README.md (this file)
2. Figshare.xlsx (Aggregate data)
3. Figshare.ipynb (Visualized code)
4. gadm41_ningbo_2.json (Map file:From GADM https://gadm.org/)

## Quick Start
Download this dataset
Install required Python packages
Run Jupyter Notebook files in order:
First, run Figshare.ipynb.ipynb
Select analysis files according to research objectives
Modify visualization parameters to generate customized charts

## Important Notes
Data have been anonymized with all personally identifiable information removed
R0 values are model estimates and should be used for reference and research purposes only
Please cite appropriately when publishing results using this dataset
